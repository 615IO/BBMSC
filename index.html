BBMSC

A simple Model / Subscriber / Controller framework for iOS

Download .zip Download .tar.gz View on GitHub

Your new favorite MSC framework.

BBMSC is meant to make using an Model / Subscriber / Controller pattern drop dead simple and fun. If you need structure around your code so that you can always rely on your views getting the data they need and you also want a simple structure that all the developers on the team can rally around then look no further. 

add BBMSC to your Podfile
$pod install

Clone the git repo if you would like to see example code on how to use BBMSC.
